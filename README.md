# Rails Tongoute API

## Install

### Clone repository

```
git clone https://github.com/fon-Luchs/Tongoute-api.git
```

### Install gems

```
gem install bundler
```

```
bundle install
```

### Create databases

```
rake db:create
```

### Run migrations

```
rake db:migrate
```

### Run specs

```
rake
```

### Run server

```
rails s
```
